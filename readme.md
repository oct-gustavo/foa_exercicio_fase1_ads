
# To-Do List

Este é um projeto que está destinado ao exercício acadêmico proposto para apresentar os principais elementos e tecnologias do Front-End. 

Por mais que seja um "projeto" simples. Agora me sinto mais confiante para iniciar algo em front-end. CRUD entre outros exemplos. 


## Tecnologias

- CSS
- HTML
- JavaScript
- IDE VS Code


## Funcionalidades

- Adiciona itens a lista.
- Marca o item como concluído.
- Desmarca como concluído o item.
- Remove o item da lista.
